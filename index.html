268 Emirps Trinomiais — uma nova família de primos reversíveis da forma 10x + c·10y + k

🧑‍🔬 Sobre o Projeto

Meu nome é Yuri Ganda e este projeto documenta a descoberta de uma família de 268 números primos reversíveis (emirps) trinomiais. O maior deles possui 6.867 dígitos. O algoritmo utiliza a biblioteca GMP com testes de Miller-Rabin e certificação via ECPP.
