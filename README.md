# Projeto LEVIATA – 268 Emirps Trinomiais

Descoberta de uma nova família de **números primos reversíveis (emirps)** da forma `10^x + c·10^y + k`.

## 🧑‍🔬 Sobre o projeto

Meu nome é **Yuri Ganda** e este é o resultado de meses de pesquisa independente. Desenvolvi um algoritmo de busca que vasculhou milhões de candidatos até encontrar **268 emirps trinomiais**. O maior deles possui **6.867 dígitos**.

Todos os números foram certificados via ECPP (usando PARI/GP e Primo) e submetidos aos bancos oficiais **Factordb** e **PrimePages**.

## 📊 Números em destaque

- **268 emirps** catalogados
- **4 a 6.867 dígitos**
- Fórmula: `10^x + c·10^y + k`
- Certificação: ECPP + PARI/GP
- Registro público: [Factordb](http://factordb.com) e [PrimePages](https://t5k.org)
